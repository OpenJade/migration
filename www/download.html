Download OpenJade and OpenSP

  • The latest stable release of OpenJade is 1.3.2.
  • The latest development release of OpenJade is 1.4devel (not available here, use CVS)
  • The latest stable release of OpenSP is 1.5.1.

Binaries

Binaries are not usually provided by the OpenJade project. Many suppliers include openjade as part of their products. This includes the major Linux and BSD suppliers. However, a Win32 binary is usually available for download as a courtesy to Windows users who may not have a suitable compiler available

If you are able to provide other useful binaries, please get in touch with the mailing list.
