OpenJade Source Control Repository Home Page

Welcome to the OpenJade CVS repository.

Jade is James Clark's implementation of DSSSL -- Document Style Semantics and Specification Language -- an ISO standard for formatting SGML (and XML) documents.

OpenJade is a project undertaken by the dsssl community to maintain and extend Jade. OpenJade is based completely on Jade, and is distributed under the same license.
