OpenJade SVN Repository Access

Like many other open source projects, OpenJade source code is controlled using Subversion. Read-only access to the repository is via one of the many standard Subverion client, or via a web browser.

To access the repository using the command-line subverion client, use

    svn co https://openjade.svn.sourceforge.net/svnroot/openjade/trunk/sp
    svn co https://openjade.svn.sourceforge.net/svnroot/openjade/trunk/jade

For more information, see SF's documentation
