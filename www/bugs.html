Reporting OpenJade Bugs

How to report bugs in OpenJade:

Submit a bug report to the Bug Tracker at the OpenJade SourceForge site. Or, send an e-mail to Openjade-devel@lists.sourceforge.net. Include:

  • The version of OpenJade which exhibits the bug (for example, 1.3pre1). This can be obtained using the -v switch.
  • The platform you are using, including compiler in case you built from source
  • A description of the bug
  • All sgml sources, stylesheets, catalogs, etc. required to reproduce the bug. Try to make a minimal set. If we can't reproduce the bug, we can't solve it, but we'd rather not wade through a 3000-line stylesheet.

At this point, OpenJade is far from being a conforming implementation of the DSSSL standard. Do not report a documented limitation as a bug.
